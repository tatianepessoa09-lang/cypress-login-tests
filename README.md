# imersao-aut-web (Cypress)

Estrutura mínima para começar com Cypress.

Como usar (PowerShell):

```powershell
cd "C:\Users\tatiane.de.s.pessoa\Desktop\Tatiane - Pessoal\Cypress\www\imersao-aut-web"
npm install
# abrir UI
npm run cypress:open
# rodar headless
npm run cypress:run
```

Arquivos criados:
- `package.json` com scripts
- `cypress.config.js` configuração mínima
- `cypress/e2e/login.cy.js` teste de exemplo
- `cypress/support/index.js` arquivo de suporte (vazio)
